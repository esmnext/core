# Snel starten

## Projectinitialisatie

Volg deze stappen om een nieuw Esmx-project aan te maken en te initialiseren:

```bash
mkdir je-app-naam
cd je-app-naam
```

## Frameworkversiekeuze

Het Esmx Framework biedt meerdere technologiestapelversies, elk geoptimaliseerd voor verschillende toepassingsscenario's. Kies de juiste versie op basis van je projectvereisten:

### HTML

Geschikt voor projecten die streven naar een minimale architectuur:

- Geen externe afhankelijkheden, direct klaar voor gebruik
- Native JavaScript-ontwikkelervaring
- Ideaal voor het bouwen van statische websites en lichtgewicht applicaties
- Ondersteuning voor geleidelijke functie-uitbreiding

[Bekijk de gedetailleerde documentatie voor de HTML-versie](../frameworks/html.md)

### Vue2

Geschikt voor de ontwikkeling van bedrijfsapplicaties:

- Volledige ondersteuning voor TypeScript
- Rijk ecosysteem van externe componenten
- Volwassen ontwikkeltoolchain
- Bewezen stabiliteit in productieomgevingen

[Bekijk de gedetailleerde documentatie voor de Vue2-versie](../frameworks/vue2.md)

### Vue3

Geschikt voor moderne webapplicaties:

- Reactief systeem gebaseerd op Proxy
- Ondersteuning voor Composition API
- Betere runtimeprestaties
- Kleinere bundelgrootte

[Bekijk de gedetailleerde documentatie voor de Vue3-versie](../frameworks/vue3.md)

### Preact+HTM

Geschikt voor projecten die streven naar lichtgewicht en hoge prestaties:

- Zeer kleine runtimegrootte (3KB)
- Native JavaScript-templatesyntaxis
- Compatibel met React API
- Uitstekende prestaties

[Bekijk de gedetailleerde documentatie voor de Preact+HTM-versie](../frameworks/preact-htm.md)

## AI-ondersteunde ontwikkeling

Het Esmx Framework biedt AI-ondersteunde ontwikkelingsmogelijkheden die de ontwikkelingssnelheid aanzienlijk kunnen verhogen:

1. Kies de juiste frameworkversiedocumentatie
2. Geef de documentatie-inhoud door aan de AI-assistent
3. De AI genereert automatisch het projectskelet en configuratiebestanden

:::tip Tip
AI-ondersteunde ontwikkeling versnelt niet alleen de projectinitialisatie, maar zorgt er ook voor dat de projectstructuur voldoet aan de beste praktijken.
:::
