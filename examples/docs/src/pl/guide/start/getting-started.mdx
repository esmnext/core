# Szybki start

## Inicjalizacja projektu

Aby utworzyć i zainicjować nowy projekt Esmx, wykonaj następujące kroki:

```bash
mkdir nazwa-twojej-aplikacji
cd nazwa-twojej-aplikacji
```

## Wybór wersji frameworka

Framework Esmx oferuje wiele wersji stosu technologicznego, z których każda jest zoptymalizowana pod kątem różnych scenariuszy aplikacyjnych. Wybierz odpowiednią wersję zgodnie z wymaganiami projektu:

### HTML

Odpowiedni dla projektów dążących do minimalnej architektury:

- Zero zależności zewnętrznych, gotowy do użycia od razu
- Natywne doświadczenie rozwoju w JavaScript
- Idealny do budowania statycznych stron internetowych i lekkich aplikacji
- Wsparcie dla stopniowego rozszerzania funkcjonalności

[Zobacz szczegółową dokumentację wersji HTML](../frameworks/html.md)

### Vue2

Odpowiedni dla rozwoju aplikacji na poziomie przedsiębiorstwa:

- Pełne wsparcie dla TypeScript
- Bogaty ekosystem komponentów stron trzecich
- Dojrzały łańcuch narzędzi deweloperskich
- Sprawdzona stabilność w środowisku produkcyjnym

[Zobacz szczegółową dokumentację wersji Vue2](../frameworks/vue2.md)

### Vue3

Odpowiedni dla nowoczesnych aplikacji webowych:

- System reaktywności oparty na Proxy
- Wsparcie dla API kompozycyjnego
- Lepsza wydajność w czasie wykonywania
- Mniejszy rozmiar pakietu

[Zobacz szczegółową dokumentację wersji Vue3](../frameworks/vue3.md)

### Preact+HTM

Odpowiedni dla projektów dążących do lekkości i wysokiej wydajności:

- Bardzo mały rozmiar środowiska uruchomieniowego (3KB)
- Natywna składnia szablonów JavaScript
- API kompatybilne z React
- Doskonała wydajność

[Zobacz szczegółową dokumentację wersji Preact+HTM](../frameworks/preact-htm.md)

## Wsparcie AI w rozwoju

Framework Esmx oferuje możliwości wsparcia AI w rozwoju, co może znacząco zwiększyć efektywność:

1. Wybierz odpowiednią dokumentację wersji frameworka
2. Przekaż treść dokumentacji do asystenta AI
3. AI automatycznie wygeneruje szkielet projektu i pliki konfiguracyjne

:::tip Wskazówka
Wsparcie AI w rozwoju nie tylko przyspiesza inicjalizację projektu, ale także zapewnia, że struktura projektu jest zgodna z najlepszymi praktykami.
:::
